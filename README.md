# vuemeteo

My frontend for meteo-station with Vue 3 in Vite.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## Deployment

- `php ../ftp-deployment/deployment deployment.ini`

## github personal token:

- pregeneruj token na: (https://github.com/settings/tokens)
- použi príkaz: `git remote set-url origin https://petak23:!!!TOKEN!!!@github.com/petak23/vuemeteo.git`
